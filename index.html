Introdução

Observando o cenário geopolítico atual, percebe-se um grande aumento dos conflitos diplomáticos ao redor do globo, percebendo o projeto visa encontrar um meio de informar o uso de armas nucleares às pessoas e ainda acredita poder criar um tipo de jogo para que os jogadores sintam por si só como é ter o poder de uma arma nuclear e como manuseá-la de forma consciente para que não aconteça prejuízos para o outro quanto para si mesmo.
